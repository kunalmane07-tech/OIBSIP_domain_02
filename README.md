Retail Sales Exploratory Data Analysis (EDA)
This project performs a complete Exploratory Data Analysis (EDA) on a retail sales dataset using
Google Colab.
The purpose of this project is to identify sales trends, customer behavior, product performance, and
insights
that can help improve business decisions.
■ Project Features
- Data Loading & Cleaning
- Handling missing values, duplicates, and incorrect formats
- Generating the TotalPrice column
- Sales trend and time series analysis
- Customer and product insights
- Visualizations using Matplotlib, Seaborn, and Plotly
- Business recommendations based on findings
■ Dataset Columns
- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country
- TotalPrice
■ Steps Performed
1. Data Import
2. Cleaning & preprocessing
3. EDA visualizations
4. Time series daily sales trend
5. Product performance analysis
6. Customer analysis
7. Insights + Recommendations
■ Visualizations Included
- Sales distribution
- Top 10 products
- Customer spending patterns
- Correlation heatmap
- Time series trend
■■ How to Run the Project
